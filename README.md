# Bengaluru Municipality Complaints Analysis

## Overview
Analyzed municipal complaint data using SQL to identify trends, high-issue areas, and gaps in resolution.

## Methodology
- Cleaned and processed raw CSV data using SQL  
- Performed analysis on complaint status, categories, wards, and locations  
- Built Power BI dashboard to visualize trends and patterns  
- Summarized insights in a 3-slide presentation  

## Key Insights
- Total complaints: 16,071  
- 55% cases are still open → high backlog  
- Transportation & sanitation issues contribute to majority of complaints  
- Complaints are concentrated in specific wards (Bellanduru, Begur)  
- Repeated issues in same areas indicate poor resolution effectiveness


## Recommendation
- Focus resources on roads and garbage-related problems. Deploy rapid response teams to high-complaint wards to handle issues faster.
- Aim to reduce the current "Open" case percentage by 20% within the next fiscal quarter through streamlined approval workflows and localized maintenance plans.
